# M1s

## Description
Sipeed dev Module with BL808 RISC-V CPUs and NPU,

see https://www.seeedstudio.com/Sipeed-M1s-Dock-AI-with-1-69-Inch-CTP-p-5545.html

- RISC-V three CPUs - BL808
- NPU - BLAI-100
- WiFi 2.4G
- BT 5
- camera: MPI, DVP
- Display: DPI, DBI, SPI
- USB 2.0 HS OTG
- MMU
